# Embodied4Arts Website (GitHub Pages)

This is a lightweight static website for **Embodied4Arts @ RSS 2026**.

## Quick deploy
1. Create repo: `embodied4arts.github.io` under the `embodied4arts` organization.
2. Upload all files in this folder to the repo root (keep folder structure).
3. GitHub → Repo Settings → Pages → Deploy from a branch → `main` / root.
4. Visit: https://embodied4arts.github.io

## Add images
- Teaser figure: `assets/img/teaser_rss.png`
- Optional people photos: put images under `assets/people/` and edit HTML if you want to display them.

## Edit content
- Main content lives in `index.html`
- Styles in `assets/css/style.css`
